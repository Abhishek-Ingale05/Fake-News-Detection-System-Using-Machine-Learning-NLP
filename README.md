# Fake News Detection System using Machine Learning & NLP

# Project Overview

The Fake News Detection System is a Machine Learning and Natural Language Processing (NLP) project used to classify news articles as:

- Fake News
- Real News

This system helps reduce misinformation by automatically analyzing news content.

---

# Objectives

- Detect fake and real news articles
- Prevent misinformation spread
- Use NLP techniques for text analysis
- Build an automated news verification system

---

# Technologies Used

## Programming Language
- Python

## Libraries
- Pandas
- Matplotlib
- Scikit-learn
- Regular Expressions (re)

---

# Machine Learning Algorithm

- Logistic Regression

---

# NLP Techniques Used

- Text Cleaning
- Lowercase Conversion
- Removing Special Characters
- TF-IDF Vectorization

---

# Dataset

Dataset contains:
- Fake News articles
- Real News articles

Files:
- Fake.csv
- True.csv

# Project Workflow

1. Data Collection
2. Data Preprocessing

# ▶️ How to Run the Project

## Step 1 : Clone the Repository

```bash
git clone https://github.com/your-username/Fake-News-Detection-System.git
```

---

## Step 2 : Navigate to Project Directory

```bash
cd Fake-News-Detection-System
```

---

## Step 3 : Install Required Libraries

```bash
pip install -r requirements.txt
```

Or install manually:

```bash
pip install pandas matplotlib scikit-learn
```

---

## Step 4 : Run the Project

```bash
python app.py
```

---

## Step 5 : Enter News Text for Prediction

Example:

```bash
Enter News Text :
```

Type any news headline or article text.

---

## Step 6 : View Prediction Result

Output:

```bash
Prediction : Fake News
```

or

```bash
Prediction : Real News
```
