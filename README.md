# Fake News Detection System using Machine Learning & NLP

# Project Overview

The Fake News Detection System is a Machine Learning and Natural Language Processing (NLP) project used to classify news articles as:

- Fake News
- Real News

This system helps reduce misinformation by automatically analyzing news content.

---

# Objectives

- Detect fake and real news articles
- Prevent misinformation spread
- Use NLP techniques for text analysis
- Build an automated news verification system

---

# Technologies Used

## Programming Language
- Python

## Libraries
- Pandas
- Matplotlib
- Scikit-learn
- Regular Expressions (re)

---

# Machine Learning Algorithm

- Logistic Regression

---

# NLP Techniques Used

- Text Cleaning
- Lowercase Conversion
- Removing Special Characters
- TF-IDF Vectorization

Dataset contains:
